Arduino Based Remote Controlled Mecanum Wheel Car. For More Information, Please Visit [yuxiliu.com](https://yuxiliu.com/portfolio/mecanum-wheel-car/)

Current Functions:
1. Omni-Direction Moving
2. Rotate
3. Remote Controllable
